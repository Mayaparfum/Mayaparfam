MAYA PARFUM - WEB ACTUALIZADA

Incluye:
- Nuevos precios de hombre
- Catálogo de mujer
- Nuevos perfumes árabes de hombre
- Buscador, filtros y carrito a WhatsApp
- Espacio para imágenes de cada perfume

CÓMO AGREGAR IMÁGENES:
1. Entra a la carpeta imagenes.
2. Guarda tus fotos en formato JPG con el nombre que aparece en cada tarjeta cuando no hay imagen.
   Ejemplos:
   imagenes/hawas-ice.jpg
   imagenes/ysl-y-edt.jpg
   imagenes/yara-candy.jpg
   imagenes/riffs-freeze.jpg

SUBIR A NETLIFY:
- Sube el ZIP completo o la carpeta completa.
- El archivo principal ya se llama index.html.
