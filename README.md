# Maya Parfum - Catálogo Web

Página lista para publicar en GitHub Pages.

## Cómo publicarla

1. Entra a GitHub y crea un repositorio nuevo.
2. Ponle nombre, por ejemplo: `maya-parfum`.
3. Sube estos archivos:
   - `index.html`
   - `README.md`
4. Ve a **Settings**.
5. Entra a **Pages**.
6. En **Build and deployment**, selecciona:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
7. Guarda.
8. Espera 1 a 3 minutos.
9. Tu página quedará en un enlace parecido a:

`https://TU-USUARIO.github.io/maya-parfum/`
